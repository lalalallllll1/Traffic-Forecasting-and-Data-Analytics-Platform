前后端分别采用JavaScript和Python，在使用时请替换为自己的文件路径和地址接口。

本系统通过使用美国加利福尼亚地区的PEMS04、PEMS07和PEMS08数据集模拟山东高速公路的交通情况，该数据集被广泛应用于交通流预测领域。具有高分辨率时间序列数据、多源传感器数据和丰富的交通特征信息等特点。并且通过百度地图API和天气数据爬取，提供了准确的地图信息和实时天气情况，提高模拟系统对实际交通情况的准确性和可靠性。下面我将对实际应用的每个模块进行具体介绍。

平台整体围绕交通数据管理、流量模式分析、热点站点识别以及预测结果展示等核心需求展开设计，实现了从基础数据可视化到预测结果分析的完整业务流程。
首先，平台提供了数据图像首页，用于对交通运行整体情况进行概览式展示，如图5.2所示。该页面集成了山东省高速公路路网示意图、关键统计指标卡片以及图表分析区域。其中，指标卡片展示了用户总数、节假日高峰流量、工作日高峰流量、最大流量区域、最大流量站点、最大流量路段以及ETC用户占比等信息；下方图表则进一步展示工作日最常出现时间与繁忙区域分布情况。该页面实现了对交通流整体态势的快速汇总，便于用户从全局视角掌握路网运行状况。

在预测结果应用方面，平台集成了本文提出的WI-ASTGCN模型，并实现了预测结果的可视化展示，如图所示。页面左侧地图以空间化方式展示不同站点的车流量等级分布，用户可通过悬浮信息查看具体站点的经纬度、车流量及更新时间等内容；右侧则通过柱状图和折线图分别展示Top10收费站流量统计和预测值、真实值对比结果。该界面表明，本文所提出的预测模型不仅能够在算法层面实现有效建模，同时也能够在平台端完成可视化部署与业务集成。

进一步地，平台还提供了预测对比分析功能。用户可自主选择时间段和站点区域，系统随即展示不同模型预测结果与真实值之间的对比曲线。该页面能够同时呈现不同区域的多组预测曲线，为模型效果评估提供直观依据。通过该功能，可以清晰观察不同方法在拟合精度、趋势跟踪能力以及局部波动刻画方面的差异，从而进一步验证所提方法在实际交通预测场景中的有效性。
